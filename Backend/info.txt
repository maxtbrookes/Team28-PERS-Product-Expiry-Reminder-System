Flask backend location, run 

    python3 app.py 

to run the file, need flask installed to run

To setup flask, please install via the command:

    pip install flask

This requires pip and python3 to be installed, if you don't have them follow this link

Python3: https://www.geeksforgeeks.org/python/download-and-install-python-3-latest-version/

Pip: Already installed with python3


For testing:
pip install pytest
python -m pytest -v
